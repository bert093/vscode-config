## Shortcut VSCode

> [!NOTE]
> Note: You can view all VSCode shortcuts on the [official website](https://code.visualstudio.com/shortcuts/keyboard-shortcuts-windows.pdf), or open all available keyboard shortcuts in VSCode by pressing the following keys: `ctrl+k ctrl+s`
> - ctrl + shift + p (command palette)
> - ctrl + alt + b (toggle secondary side bar, default -> chat with ai agent)
> - ctrl + j (toggle panel / join lines)
> - ctrl + b (toggle primary sidebar)
> - ctrl + . (open vscode settings)
